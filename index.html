🟣 Nostr Web Apps – iframe Loading Test

Each card attempts to load a known Nostr web app inside an <iframe>. A green badge means the site allowed embedding; a red badge means it was blocked (usually via X-Frame-Options or Content-Security-Policy).

⏳ Loading – waiting for response
✅ Allowed – iframe loaded successfully
